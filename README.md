# Personal Portfolio

Personal portfolio website built to showcase my projects, experience, and skills as a software engineer.

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local Development

### Prerequisites
- Node.js (18+ recommended)
- npm

### Setup
```bash
git clone https://github.com/<your-username>/rithu-portfolio.git
cd rithu-portfolio
npm install
npm run dev
